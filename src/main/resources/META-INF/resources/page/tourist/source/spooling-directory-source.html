In which order files in the spooling directory will be consumed oldest,youngest and random. In case of oldest and youngest, the last modifiedtime of the files will be used to compare the files. In case of a tie, the filewith smallest lexicographical order will be consumed first. In case of random anyfile will be picked randomly. When using oldest and youngest the wholedirectory will be scanned to pick the oldest/youngest file, which might be slow if thereare a large number of files, while using random may cause old files to be consumedvery late if new files keep coming in the spooling directory.
