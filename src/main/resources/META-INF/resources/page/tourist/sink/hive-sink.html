Hive grants a batch of transactions instead of single transactions to streaming clients like Flume.This setting configures the number of desired transactions per Transaction Batch. Data from alltransactions in a single batch end up in a single file. Flume will write a maximum of batchSize eventsin each transaction in the batch. This setting in conjunction with batchSize provides control over thesize of each file. Note that eventually Hive will transparently compact these files into larger files.
